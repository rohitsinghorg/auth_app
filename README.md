# Auth UI App

An Authentication UI project

## Getting Started

This project has login and OTP Screen UI.

## Structure

We are using the below directory structure for this project:

auth_app
├── assets
│ ├── fonts
│ ├── images
│ │ ├── login.png
│ │ └── otp.png
│ └── screenshots
└── lib
├── extension
│ └── context_extension.dart
├── screens
│ ├── login_screen.dart
│ └── otp_screen.dart
├── utils
│ ├── local_images.dart
│ └── strings_en.dart
├── widget
│ ├── otp_field_widget.dart
│ └── primary_button_widget.dart
└── main.dart
